# ■ Shell Scripting — Learning Notes

## ■ What Is This Section?

This folder contains my Shell Scripting learning notes from Abhishek DevOps course.

Shell scripting helps automate repetitive tasks in Linux systems.
It is widely used by DevOps engineers for automation, monitoring, and infrastructure management.

This section documents:

* Commands learned
* Scripts created
* Real-world use cases
* Hands-on automation

---

## ■ Why Shell Scripting?

Shell scripting is important for DevOps engineers because:

* Automates manual tasks
* Helps manage servers
* Used in CI/CD pipelines
* Useful for monitoring systems
* Used in infrastructure automation
