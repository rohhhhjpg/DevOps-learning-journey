## ■ Notes for today (If Any)
DevOps Definition.

DevOps is a process of improving the application delivery by ensuring there is a proper automation with a code quality that is maintained, with proper quality, continuous monitoring and testing. 

Why DevOps?

10 years back to deliver a application there were lot of teams involved like to deliver a application from developers laptop to customer,
System administrator creates a app server, where the code runs and application works after this Build and release engineer will continuously test the application and finally uploads the app in the production server.

To minimalize this DevOps culture was introduced. To improve the delivery of the application.
