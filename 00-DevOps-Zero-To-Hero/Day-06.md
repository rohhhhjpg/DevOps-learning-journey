What is an operating system?

Its acts an an bridge between hardware and software. It is the heart of the system.

Why Linux used more than windows in production environment?

Linux is fast and more secured.
It is an open sorce and it has so many distributions.
ex: Ubuntu,Debian etc.

What is kernel?

Kernel is the heart of an operating system.

It has four responsibilities.
- Device management
- Memory management
- process management
- Handling system files.

These are all the main aspects for operating system which is handled by kernel.

System libraries
It is responsible for performing a task.
Libc is one of the system library.

## ■ Commands Practiced (If Any)

- ls - List files
- pwd - present working directory
- cd - Change directory
- cd .. - Go one back directory
- cd../.. - Go two back drectory
- ls -ltr - list the files and folders with its properties
- touch - create a file
- vi - create a file and write inside a file.
- cat - print the file
- mkdir - create directory
- rm - removing files
- rm -r - removing directory
- free -g - Memory of your server
- nproc - list number of cpu's
- df -h - Disksize
- top - complete information in once place. (memory , cpu's, disksize - everything in one place)
