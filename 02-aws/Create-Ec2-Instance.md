## ■ Notes

- Select Ec2 service.
- Click on create instance
- There give name of the instance
- Select the software Ex: Ubuntu
- Select the machine CPU etc.
- Key pair value is mandatory to login that virtual machine
- So create a key pair with some name 
- .pem file is for ssh, where another one is for putty

- Open MobaXterm 
- Now login AWS instance and copy the public IPV4 Ip address
- Specify name as ubuntu
- Paste it in MobaXterm
- Now in key pair value search it in folder and select it.
- Now give run
- Now the instance you created in AWS we can able to use it and run our commands over here.
